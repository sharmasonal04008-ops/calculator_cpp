# Simple Calculator in C++

A beginner-friendly command-line calculator built using C++. It performs basic arithmetic operations based on user input.

## Features

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Modulus (%)

## Technologies Used

- C++
- Standard Input/Output (`iostream`)

## How It Works

1. Enter the first number.
2. Choose an operator (`+`, `-`, `*`, `/`, `%`).
3. Enter the second number.
4. The program displays the result.

## Example

```text
n1: 10
+,-,*,/,%
+
n2: 20
30
```

## Code Concepts

- User Input
- Conditional Statements (`if`)
- Arithmetic Operators
- Basic C++ Programming

## How to Run

1. Compile the program:
   ```bash
   g++ calculator.cpp -o calculator
   ```

2. Run the executable:
   ```bash
   ./calculator
   ```
